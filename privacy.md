Privacy Policy for RoSound II

This privacy policy explains how RoSound II handles user data.

1. Data Collection
RoSound II may collect basic Discord data such as:
- User IDs
- Server IDs
- Command usage

2. Data Usage
The collected data is only used to operate bot features and improve functionality.

3. Data Sharing
RoSound II does not sell, trade, or share user data with third parties.

4. Data Storage
Data may be stored securely on servers used by the developer.

5. Data Removal
Server owners may request data removal by contacting the developer.

6. Changes
This privacy policy may be updated at any time.

7. Contact
If you have questions about this policy, contact the developer.
