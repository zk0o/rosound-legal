Overview

Our Commitment to Privacy

This Privacy Policy describes how RoSound II collects, uses, and handles information when you use the Bot. We are committed to handling your data responsibly and transparently. By using the Bot, you consent to the practices described in this policy.

01

Data We Collect

To operate its features, the Bot may collect and temporarily store the following:

  • Discord User IDs and Server IDs
  • Roblox account cookies provided voluntarily by users
  • Upload operation IDs and associated audio metadata
  • Command usage data for feature functionality
  • Channel IDs used for result notifications
02

Sensitive Credentials

When you register a Roblox cookie with the Bot, that credential is stored locally on the Bot's server solely to perform upload operations on your behalf. You acknowledge that:

  • Cookies are stored in encrypted or access-restricted files on the Bot's server.
  • We do not use your credentials for any purpose other than the features you explicitly request.
  • You may remove your stored cookies at any time using the /removecookie command.
  • We strongly recommend rotating or invalidating your Roblox cookies periodically.
03

How We Use Data

Collected data is used exclusively to:

  • Perform audio uploads to Roblox on your behalf.
  • Track upload operation statuses and deliver results.
  • Resume pending operations after Bot restarts.
  • Improve Bot stability and features over time.
04

Data Sharing

RoSound II does not sell, trade, rent, or share your personal data with any third parties. The only external service your data is sent to is the Roblox API — solely for the purpose of performing upload requests that you initiate.

05

Data Retention

Upload operation data is retained in local files on the Bot's server for a limited period to ensure accurate result delivery. Pending operation data is automatically removed once results are confirmed or after a defined expiry period. Cookie data persists until you remove it via /removecookie.

06

Data Removal

You may request the removal of your stored data at any time by:

  • Using /removecookie to delete registered credentials.
  • Contacting the developer through the support server to request full data deletion.

Access Requirements

To use RoSound II, you must either be a member of the official RoSound support server, or have your server whitelisted by an admin. Servers not in the whitelist will not be able to interact with any bot commands. Contact the developer through the support server to request whitelist access.

Public Commands
/upload Upload MP3 or OGG audio files directly to Roblox using your registered account cookies. Supports up to 10 simultaneous uploads with automatic metadata randomization to avoid duplicate detection. Results are delivered via DM. public
/download roblox Download any Roblox audio asset by ID. For private or restricted assets, RoSound uses proprietary internal methods to bypass access restrictions and retrieve the audio — no public API is used for this process. secret method
/download soundcloud Download audio from any public SoundCloud track URL. public
/download youtube Download audio from any YouTube video by ID or full URL. public
/bypass Process an audio file using one of three bypass methods: Long Bait (appends start and end bait audio), Copyright (applies audio filters to modify the fingerprint), or Angel Society Method (concatenates a special bait using the Tpos 203 technique). Output in OGG format. public
/audio effect Apply audio effects to an uploaded file (reverb, echo, pitch shift, and more). public
/loud Apply LoudAsset processing to an audio file to make it significantly louder for Roblox use. public
/intro Merge a custom intro file with any audio file into a single output. public
/cookie Register a Roblox account cookie (.ROBLOSECURITY) to enable uploads from that account. Supports multiple accounts. public
/removecookie Remove a previously registered Roblox cookie. Shows account name, validity status, and remaining upload quota for each registered account. public
/operation Check the live moderation status of any Roblox upload operation by ID. Fetches real-time data from the Roblox API using your stored credentials. public
Admin / Owner Commands
/serveradmin Manage the server whitelist database. Subcommands: add/remove servers, configure allowed channels per server, and download the full servers database file. admin only
/gentts Generate a TTS (text-to-speech) audio using AI with selectable voices and languages. admin only
/genttsandmix Generate AI TTS audio and optionally mix it with a music file. Supports language selection, voice selection, and music cut timing. admin only
/addaudio Upload a new Angel Society bait audio file to the bot's bait library. Owner only. owner only
/addbait Add new long bait audio files (start and end segments) to the bot's bait collection used by the /bypass command. admin only
/library Send the RoSound audio library panel — search, manage, and organize Roblox audio IDs by creator name, song title, or asset ID. admin only
